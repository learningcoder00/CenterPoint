# 前端主题切换改动说明

## 一、改动目标

本次改动为 Vue 前端增加了全局主题切换能力，支持：

- 黑夜模式
- 白天模式

主题切换按钮放置在页面右上角，`Clips` 页和 `Results` 页都可以使用。

## 二、涉及文件

- `App.vue`
- `assets/main.css`
- `views/ClipsView.vue`
- `components/ClipCard.vue`
- `components/PreviewModal.vue`

## 三、主要改动内容

### 1. `App.vue`

- 新增全局主题切换按钮
- 使用 `localStorage` 记忆用户上次选择的主题
- 页面重新打开后自动恢复上次使用的模式
- 将切换图标调整为简约线性风格的月亮 / 太阳图标

### 2. `assets/main.css`

- 增加黑夜模式与白天模式两套主题变量
- 统一页面背景、卡片、输入框、按钮、弹窗、导航等区域的主题配色
- 增加主题切换过渡效果
- 补充白天模式下主按钮、悬浮条、预览工具栏等区域的专用颜色变量

### 3. `components/ClipCard.vue`

- 修复 `clip` 卡片鼠标悬停播放时右下角帧数提示在白天模式下的显示问题

### 4. `components/PreviewModal.vue`

- 修复单个 `clip` 预览弹窗底部播放栏在白天模式下的显示问题
- 调整白天模式下 `保存 tags` 按钮的视觉样式，使其更清晰、更协调

### 5. `views/ClipsView.vue`

- 修复 `Clips` 页面底部悬浮批量操作条在白天模式下未适配的问题
- 让 `开始可视化` 和 `取消选择` 按钮在白天模式下与整体页面风格保持一致

## 四、当前新增功能

### 1. 主题切换

用户可以通过右上角按钮在黑夜模式和白天模式之间切换。

### 2. 主题记忆

刷新页面或下次重新打开页面时，会自动恢复上次使用的主题模式，不再固定回到默认黑夜模式。

### 3. 白天模式细节补全

本次补全了以下几个白天模式下容易遗漏的区域：

- `Clips` 卡片悬停播放时的帧数提示
- 预览弹窗底部播放栏
- `保存 tags` 按钮
- `Clips` 页面底部悬浮批量操作条

## 五、Windows 脚本说明

- `start_frontend.cmd`：在 Windows 下启动前端开发服务器，用于本地实时预览 Vue 页面。
- `build_frontend.cmd`：在 Windows 下打包前端代码，生成给后端使用的 `dist` 静态文件。
- `start_backend.cmd`：在 Windows 下启动 Java 后端服务，并通过后端地址访问项目页面和接口数据。
